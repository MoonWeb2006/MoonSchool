Destinations

What We Teach?

Web development

Web programming is a simple but very popular field.
During the training, you will learn the basics of web design and create simple but attractive websites.
This direction is suitable for beginners

IoT

The Internet of Things is a concept that involves the use of a huge number of devices (things)
that interact not only with a person, but also with each other, as well as with other information systems.

VR / AR

VR / AR is a rapidly developing field in the field of information
technology.Remember Pokemon go and you will understand everything.During
the course, you will learn the Blender program and learn how to work with the
Unity game engine.

Graphic Design

Graphic design is the best way to convey the creator's thoughts to the user of the
product.During the course, you will get acquainted with graphic editors, such as
Photoshop and Figma.
